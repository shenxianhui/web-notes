# Boolean
Boolean 是对应布尔值的引用类型。要创建一个 Boolean 对象，就使用 Boolean 构造函数并传入 `true` 或 `false`，如下例所示：

```
let booleanObject = new Boolean(true);

// 实际开发中, 我们一般这样写:
let myBoolean = true;
```
