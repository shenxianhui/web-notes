# HTML 格式化标签
HTML 使用标签 `<b>`("bold") 与 `<i>`("italic") 对输出的文本进行格式, 如：**粗体** or *斜体*

这些HTML标签被称为**格式化标签**
```
<b>加粗文本</b>
<br />
<i>斜体文本</i>
<br />
<del>删除文本</del>
```
